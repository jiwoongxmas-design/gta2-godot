# GTA2 Godot Clone

**Godot Engine 4.x 기반 Top-Down 2D 액션 게임**

## 🎮 게임 소개
- 📍 Top-Down 뷰의 오픈월드 게임
- 🚗 차량 조종 시스템
- 🎯 전투 및 무기 시스템
- 👥 NPC와 경찰 AI
- 💰 미션 시스템

---

## 📦 프로젝트 구조

```
gta2-godot/
├── scenes/              # 게임 씬

│   ├── npcs/            # NPC 씬
│   ├── weapons/         # 무기 씬
│   ├── levels/          # 레벨/맵

├── scripts/             # GDScript 코드
│   ├── player/
│   ├── vehicles/
│   ├── ai/
│   ├── weapons/
│   ├── ui/

│   └── music/

```

## 🚀 시작하기

### 1. Godot 4.x 설치
- [godotengine.org](https://godotengine.org) 에서 다운로드

### 2. 프로젝트 열기
```bash
# 이 리포 클론
git clone https://github.com/jiwoongxmas-design/gta2-godot.git
cd gta2-godot

# Godot 에디터에서 폴더 열기
```

### 3. 실행
- Godot 에디터에서 Play 버튼 클릭
